# 🌦️ Weather-Cast Dashboard

A responsive weather dashboard built using HTML, CSS, and Vanilla JavaScript that provides real-time weather information using the OpenWeather API.

## 🚀 Features

### 🌍 Real-Time Weather Data
- Fetches live weather information using OpenWeather API.
- Displays:
  - Temperature
  - Weather Condition
  - Humidity
  - Wind Speed
  - Pressure
  - Feels Like Temperature

### 🔍 City Search
- Search weather information for any city worldwide.
- Supports search using button click and Enter key.

### 📍 Geolocation Support
- Automatically detects user's current location.
- Displays weather data based on browser location.

### 💾 Local Storage Caching
- Stores recently searched city weather data.
- Uses cached data for up to 10 minutes to reduce API calls.

### 🎨 Dynamic Weather Themes
- Sunny Theme
- Cloudy Theme
- Rainy Theme

Theme automatically changes based on current weather conditions.

### 🌙 Dark Mode
- Dark/Light mode toggle.
- User preference persists using LocalStorage.

### 📱 Responsive Design
- Mobile-friendly layout.
- Glassmorphism UI design.
- Optimized for desktop, tablet, and mobile devices.
